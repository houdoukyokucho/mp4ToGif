# Product
    convertMp4ToGif

# Feature
    同階層にある全ての mp4 を gif に変換します。

# environment
    windows10

# requirements
    ffmpeg（https://fukatsu.tech/windows-ffmpeg）

# Usage
1.convertMp4ToGif.bat と同階層に mp4 ファイルを置く。
2.convertMp4ToGif.bat をクリックする。

# tree
    -

# Note
    もとの mp4 ファイルが破棄される事。
    ffmpeg インストール後は環境変数の設定をする事。
